WorkArena: How Capable Are Web Agents at Solving Common Knowledge Work Tasks?

Alexandre Drouin, Maxime Gasse, Massimo Caccia, Issam H. Laradji, Manuel Del Verme, Tom Marty, Léo Boisvert, Megh Thakkar, Thibault Le Sellier De Chezelles, Quentin Cappart, Nicolas Chapados, Alexandre Lacoste
Paper Code arXiv

Abstract

We study the use of large language model-based agents for interacting with software via web browsers. Unlike prior work, we focus on measuring the agents' ability to perform tasks that span the typical daily work of knowledge workers utilizing enterprise software systems. To this end, we propose WorkArena, a remote-hosted benchmark of 33 tasks based on the widely-used ServiceNow platform.

We also introduce BrowserGym, an environment for the design and evaluation of such agents, offering a rich set of actions as well as multimodal observations. Our empirical evaluation reveals that while current agents show promise on WorkArena, there remains a considerable gap towards achieving full task automation. Notably, our analysis uncovers a significant performance disparity between open and closed-source LLMs, highlighting a critical area for future exploration and development in the field.

Benchmark Overview

Task Structure

33 Task Templates
Basic navigation and simple operations within ServiceNow interface. Foundation skills including form filling, record creation, and basic searches.

  • Max Steps: 30
  • Seed Diversity: High
  • Multi-tab: No
  • Focus: Basic Operations

Enterprise Domains

IT Service Management

  • Incident management and resolution
  • Change request processing
  • Asset management and tracking
  • Service catalog operations

Human Resources

  • Employee onboarding processes
  • Leave request management
  • Performance tracking
  • Organizational structure updates

Customer Service

  • Case management workflows
  • Knowledge base maintenance
  • SLA compliance tracking
  • Customer communication

Administrative Tasks

  • User access management
  • System configuration
  • Report generation and analysis
  • Data import/export operations

Key Features

High Seed Diversity

Each task template includes multiple seed variations, ensuring robust evaluation across different data scenarios and preventing overfitting.

Demo Instance Hosting

Tasks run on a live ServiceNow demo instance, providing authentic enterprise software interactions without complex setup requirements.

Foundational Tasks

33 foundational task templates covering essential enterprise operations including navigation, form filling, record creation, and basic searches.

Real Enterprise Context

Tasks reflect actual enterprise workflows and business processes, providing realistic evaluation of agent capabilities in workplace settings.
